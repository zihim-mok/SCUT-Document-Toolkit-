
个人整理的华南理工大学（华工）相关素材与常用模板，便于课程汇报、报告撰写与演示排版统一。


## 使用说明

- **PPT 模板**：用 PowerPoint / WPS 打开对应 `.pptx`，按需替换文案与配图；若版本较旧，建议「另存为」当前软件支持的格式后再编辑。
- **课程设计报告书模板**：用 Word / WPS 打开 `.doc`，按课程要求填写章节；导出提交前请核对院系最新格式要求。
- **校徽**：可直接插入文档或 PPT；印刷或对外展示前请确认是否符合学校官方标识使用规范。
